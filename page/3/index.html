用 VirtualBox 建立 Max OSX 環境

發表於 2013-05-06 | 分類於 IT 筆記

VirtualBox是個方便好用的虛擬機器程式，
前陣子為了要建立一個OS X的環境耗了塊一整天

留個紀錄讓未來的自己看看自己有多渣總是個好的(喂)

參考文章
http://www.lifehacker.com.au/2012/08/how-to-run-mac-os-x-on-any-windows-pc-using-virtualbox/

Step 1 材料準備

  1. VirtualBox 4.2.6 for windows host
  2. Hackboot 開機iso Hackboot1 Hackboo2
  3. Mac作業系統iso&nbsp

GuestAddtions不支援OS X
所以當Guest雙向剪貼簿、雙向檔案傳輸都無法使用
基本上要使用方便要另外的用

  • 共享資料夾
  • 更改解析度這兩個方法

Step 2 共享資料夾

連線至 smb://10.0.2.2
即為host的share資料夾

參考http://www.takwing.idv.hk/tech/virtual/faq/access_shared_folder.html

Step 3 更改解析度

方法1 由 Hackboot 開機畫面輸入參數修改解析度

方法2 讓 OS X 開機時直接修改解析度
在OS X底下的
/Library/Preferences/SystemConfiguration/com.apple.boot.plist這個xml底下

增加
Kernel Flags
"GraphicsMode"="1360x768x32"

重點：Graphics Mode 中間請不要加空白
