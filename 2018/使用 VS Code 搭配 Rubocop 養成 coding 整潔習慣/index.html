[Rails] 使用 Visual Studio Code 搭配 Rubocop 養成 coding 整潔習慣

基本上，新手寫code總是有些不乾淨的習慣
像是多加空白啦、少加空白啦 縮排沒有對齊等等

Rubocop 就是 在 Ruby 環境中出現的一個工具，可以幫助你檢查有哪些地方不符標準

最原生的作法就是安裝後在檔案存放目錄下執行 rubocop 指令

這樣會出現像是單元測試的檢查結果(如下圖)
到這個部分還是沒什麼用

但是搭配特定文字編輯器 就可以直接在 IDE 中偵測、檢查相關異常
在編輯檔案的時候就直接順手養成良好慣例

綠色底線 像在 Word 中出現的拼字檢查一樣

—

如何在 Visual Studio Code 中安裝、使用 Rubocop

  1. 安裝 Visual Studio Code (省略步驟

  2. 安裝 rubocop

    gem install rubocop

  3. 設定 Visual Studio Code 與 rubocop 連結

在 Extension 列表中搜尋 rubocop

並點選安裝、安裝完成後點選重新載入

點選安裝

點選重新載入

設定 Setting File

點選 [Code] -> [喜好設定] -> [設定]

點選 Setting File — settings.json

加上以下兩行

"ruby.rubocop.executePath": "",

"ruby.rubocop.onSave": true

這樣就大功告成啦！
