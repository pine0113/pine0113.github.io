[Linux] 指令 — dd

data duplicator 低階資料處理工具

基礎作法
dd if=[your_input_file] of=[your_output_file]

—

直接複製整顆硬碟

sudo dd if=/dev/sda of=/dev/sda conv=noerror,sync

使用 watch 顯示 dd 進度

watch -n 5 killall -USR1 dd

空間參數

bs=[BYTES] // 
count=N

ibs=BYTES
obs=BYTES

seek=N
skip=N

conv=CONVS // [ascii,block,lcase,ucase...]

ex

dd if=/dev/sda of=/dev/sdg bs=1G count=22 conv=noerror,sync
