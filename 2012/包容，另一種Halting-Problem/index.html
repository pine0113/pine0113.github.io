包容，另一種Halting Problem

發表於 2012-11-11 | 分類於 日記

準備計算機理論期中的時候複習到 Halting Problem，
想起當初準備研究所的時候就想到的例子。

##自以為證明「歧視是不可能有消除的一天」(喂)。##

假設一個不歧視別人的人嘗試消除已經存在的「歧視」：

  1. 「沒有歧視」，是去接受世界上所有的人的想法
  2. 世界上已經存在「歧視」別人的人
  3. 根據1、2，沒有歧視的人必須接受「歧視別人的人」的想法
  4. 因為接受「歧視別人的人」的想法，所以接受歧視→歧視不會消除
    如果要消除歧視的存在，
    則就是「歧視」「歧視別人的人」，產生矛盾

故，「歧視是不可能有消除的一天」

就跟 為了要讓「殺人」這件事情不存在
就先把「會殺人的人」先殺了一樣

人總是有主觀、立場的，
一旦立場站定，就會有站在相反立場的人。
不是對錯，而是位置問題。

每個人站在那個位置，都會有同樣的立場

如果為了「不歧視別人」、「不和別人對立」
而不想要站在一個立場上。

是件愚蠢的事情，但我其實常常幹。
是要自覺的去修正了吶。
